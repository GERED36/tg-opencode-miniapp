# Telegram Bot for OpenCode (Multi-User)

Telegram-бот, связывающий пользователей с OpenCode на их ПК.

## Возможности

- Любой пользователь может подключить свой ПК к боту
- Бот выполняет команды OpenCode на ПК пользователя
- Отправка текстовых и голосовых сообщений
- Маршрутизация команд по пользователям

## Как это работает

```
Пользователь A (PC)          Бот (ваш ПК)           Пользователь B (PC)
┌─────────────────┐         ┌──────────────┐         ┌─────────────────┐
│ agent.py        │◄───────►│  bot.py      │◄───────►│  agent.py       │
│ :9000           │  TCP    │  маршрутизация│  TCP    │  :9000          │
│ opencode run... │         │              │         │  opencode run... │
└─────────────────┘         └──────────────┘         └─────────────────┘
```

## Требования

- Python 3.10+
- FFmpeg (для конвертации аудио)
- OpenCode CLI (на ПК каждого пользователя)
- Telegram Bot Token (от @BotFather)

## Установка

### 1. Установите FFmpeg

**Windows:**
```bash
winget install Gyan.FFmpeg
```

**macOS:**
```bash
brew install ffmpeg
```

**Linux:**
```bash
sudo apt install ffmpeg
```

### 2. Установите Python-зависимости

```bash
pip install -r requirements.txt
```

### 3. Настройте переменные окружения

Отредактируйте файл `.env`:

```
BOT_TOKEN=ваш_токен_бота
```

### 4. Запустите бота

```bash
python bot.py
```

## Подключение ПК пользователя

### На ПК пользователя:

1. Скопируйте файлы `agent.py` и `requirements.txt`
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Убедитесь, что `opencode` установлен и настроен
4. Запустите агент:
   ```bash
   python agent.py --port 9000
   ```
5. Скопируйте **токен** из вывода
6. Узнайте свой внешний IP (whatismyip.com)

### В чате с ботом:

1. Отправьте `/connect <ваш_ip>:<порт>`
   ```
   /connect 1.2.3.4:9000
   ```
2. Вставьте токен агента
3. Готово! Пишите сообщения — бот будет выполнять на вашем ПК

## Команды бота

| Команда | Описание |
|---------|----------|
| `/connect <ip>:<port>` | Подключить ПК |
| `/disconnect` | Отключить ПК |
| `/myip` | Моё подключение |
| `/sessions` | Список сессий |
| `/session <id>` | Продолжить сессию |
| `/new` | Новая сессия |
| `/models` | Список моделей |
| `/stats` | Статистика токенов |
| `/help` | Справка |

## Аргументы agent.py

```
python agent.py [--port PORT] [--work-dir DIR] [--token TOKEN]
```

| Аргумент | По умолчанию | Описание |
|----------|--------------|----------|
| `--port` | `9000` | Порт для прослушивания |
| `--work-dir` | Текущая папка | Рабочая директория opencode |
| `--token` | Авто-генерация | Токен авторизации |

## Структура файлов

```
├── bot.py              # Центральный бот (ваш ПК)
├── agent.py            # Агент (на ПК пользователя)
├── config.py           # Конфигурация бота
├── stt.py              # Распознавание голоса
├── requirements.txt    # Зависимости Python
├── .env                # Секреты (не коммитить!)
├── connected_agents.json  # Подключённые агенты
└── README.md           # Этот файл
```

## Безопасность

- Агент принимает команды только с правильным токеном
- Токен генерируется при каждом запуске агента
- Только админ может видеть список подключённых агентов
- Соединение по TCP (для продакшена рекомендуется SSH-туннель)
